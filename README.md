# SNAP CAM — Twitch Channel Points Overlay

Портативное приложение для стримеров: медиа-оверлей для OBS и воспроизведение треков из Яндекс Музыки по Channel Points.

---

## Установка

1. Скачай последний `twitch-helper.exe` со страницы [Releases](../../releases)
2. Положи `.exe` в любую папку — установка не нужна
3. Запусти и следуй инструкциям авторизации

---

## Быстрый старт

### Авторизация Twitch
1. Открой приложение → экран "Войти через Twitch"
2. Нажми кнопку — откроется браузер, войди в аккаунт стримера
3. После авторизации вернись в приложение

### Яндекс Музыка (опционально)
1. Перейди в раздел "Настройки"
2. Нажми "Войти через Яндекс" — откроется браузер с кодом подтверждения
3. Введи код на странице ya.ru/device

### OBS Browser Source
Добавь в OBS источник **Browser Source** с URL:
```
http://127.0.0.1:7891/overlay/default
```
Рекомендуемые настройки: 1920×1080, "Shutdown source when not visible" — OFF

---

## Функции

- **Медиафайлы** — воспроизводит видео/гифки поверх стрима по Channel Points
- **Мемы** — случайный файл из папки
- **Яндекс Музыка** — стример называет трек или вставляет ссылку, музыка играет в оверлее с анимацией
- **Группы оверлеев** — разные слоты можно направить в разные Browser Source
- **Очередь** — одновременно воспроизводится только один слот

---

## Конфигурация

При первом запуске создаётся `config.json` рядом с `.exe`. Редактировать вручную не нужно — всё настраивается через UI.

Токены Twitch и Яндекс хранятся в Windows Credential Manager (безопасно).
