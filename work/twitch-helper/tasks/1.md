# Task 1: Project Infrastructure

## Required Skills

Перед выполнением задачи загрузи:

## Description

Инициализируем Electron 33+ проект с TypeScript, React-рендерером и Vite-бандлером. Настраиваем electron-builder для portable `.exe`, `@electron/rebuild` для нативных модулей (robotjs, keytar), Jest для тестов, ESLint + Prettier, и структуру папок.

Это основа всего проекта — без неё ни одна другая задача не может быть выполнена.

## What to do

1. Создать `package.json` с зависимостями: electron v33+, react, react-dom, robotjs@0.7.0, keytar@7.9.0, @electron/rebuild, express, ws, electron-builder, pkce-challenge, uuid, vite, @vitejs/plugin-react, typescript, jest, @types/*, eslint, prettier
2. Создать `tsconfig.json` с настройками для main (Node) и renderer (DOM) процессов
3. Создать `vite.config.ts` — два entry points: `src/renderer/index.html` и `src/overlay/index.html`
4. Создать `electron-builder.config.js` с portable target для Windows x64, `PORTABLE_EXECUTABLE_DIR` env var
5. Добавить в `package.json` скрипты: `dev`, `build`, `test`, `rebuild` (`electron-rebuild`)
6. Создать структуру папок: `src/main/`, `src/main/twitch/`, `src/main/store/`, `src/main/queue/`, `src/main/actions/`, `src/main/overlay/`, `src/main/snap/`, `src/main/ipc/`, `src/main/slots/`, `src/renderer/`, `src/renderer/screens/`, `src/renderer/components/`, `src/overlay/`
7. Создать минимальный `src/main/index.ts` (точка входа Electron main process)
8. Создать минимальный `src/renderer/index.html` и `src/renderer/App.tsx`
9. Создать минимальный `src/overlay/index.html`
10. Настроить `.gitignore`: node_modules, dist, *.key, .env, auth.json, config.json
11. Настроить Jest: `jest.config.ts`, поддержка TypeScript через ts-jest
12. Запустить `npm run rebuild` — подтвердить что robotjs и keytar собираются под Electron ABI

## Details

**Files:**
- `package.json` — все зависимости и скрипты
- `electron-builder.config.js` — portable Windows x64 target; `PORTABLE_EXECUTABLE_DIR` env must be set by electron-builder
- `vite.config.ts` — multi-entry: renderer + overlay
- `tsconfig.json` — strict mode, paths для main/renderer/overlay
- `.gitignore` — node_modules, dist, config.json, *.key, .env

**Dependencies:**
- Electron 33 требует `@electron/rebuild` после любого обновления версии
- robotjs v0.7.0 — нативный модуль, собирается через node-gyp
- keytar v7.9.0 — нативный модуль, archived Dec 2022 but prebuilds exist for Electron 33

**Edge cases:**
- Если `electron-rebuild` падает — проверить python3 и node-gyp в PATH
- Для Windows нужен Visual Studio Build Tools (C++ компилятор)

**Implementation hints:**
- `electron-builder.config.js`: `target: "portable"`, `artifactName: "twitch-helper.exe"`
- В `package.json` скрипт `rebuild`: `"electron-rebuild -f -w robotjs,keytar"`
