# Task 8: Mask Action Handler

## Required Skills

Перед выполнением задачи загрузи:

## Description

При выкупе маски: симулировать нажатие hotkey через robotjs, подождать 30 секунд, симулировать нажатие глобального "снять маску" hotkey из config. При успехе — fulfill redemption. При ошибке robotjs — cancel redemption. Оба результата пишутся в лог через IPC.

## What to do

1. Создать `src/main/actions/hotkey-parser.ts` — `parseHotkey(hotkey: string)`:
   - Парсить строку типа `"ctrl+shift+1"` → `{ key: '1', modifiers: ['control', 'shift'] }`
   - Маппинг: `ctrl` → `control`, `alt` → `alt`, `shift` → `shift`, `win` → `command`
   - Пустая строка → throw `Error('Hotkey cannot be empty')`
   - Неизвестная клавиша → throw `Error('Unknown key: X')`
2. Создать `src/main/actions/mask.ts` — `MaskAction`:
   - `execute(slot: MaskSlot, redemption: RedemptionEvent): Promise<void>`
   - Последовательность: `robot.keyTap(key, modifiers)` (apply mask) → `setTimeout(30000)` → `robot.keyTap(removeMaskKey, removeMaskModifiers)` (remove mask) → `fulfillRedemption()`
   - `config.removeMaskHotkey` — глобальный hotkey, применяется для всех mask-слотов
   - При ошибке robotjs на любом шаге → `cancelRedemption()` → emit error log
   - Emit log entry через IPC: `{ viewerName, rewardTitle, status, errorMessage? }`
